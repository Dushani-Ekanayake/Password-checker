# Password-checker

This project is a comprehensive password strength checker web application that provides:

- Real-time password strength feedback with a mood meter
- AI-based smart suggestions for improving password security
- Real-world password breach checking using the Have I Been Pwned API
- Password visibility toggle for better user experience
- Hacker-style modern UI with responsive design

## Features

- Evaluates password complexity based on length, uppercase, lowercase, digits, and special characters.
- Provides actionable suggestions to improve password strength.
- Checks if the password has appeared in known data breaches to warn users.
- User-friendly interface with interactive visual feedback.

## Technologies Used

- HTML5, CSS3 (with glassmorphism and neon glow effects)
- JavaScript (ES6+)
- [CryptoJS](https://github.com/brix/crypto-js) for SHA-1 hashing
- Have I Been Pwned API for breach checking

## Usage

1. Clone or download the repository.
2. Open `index.html` in a modern browser.
3. Enter a password to see strength evaluation, suggestions, and breach status.
